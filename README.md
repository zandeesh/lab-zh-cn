# lab-zh-cn

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相对独立，不针对任何单一域名或网站，适合作为轻量级的前端实验、工具页面或静态内容存档。

## 目录说明

```
.
├── index.html          # 仓库主页 / 导航页
├── pages/              # 各独立 HTML 页面存放目录
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
└── assets/             # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：每个 HTML 文件为一个独立页面，彼此之间无强依赖关系。
- `assets/`：存放页面共用的样式、脚本和媒体文件，便于统一维护。

## 页面归档说明

所有页面均以纯 HTML + CSS + JavaScript 实现，不依赖后端服务。每个页面可单独访问，也可通过仓库主页进行导航。

归档内容涵盖：

- 前端功能演示
- 交互原型
- 信息展示页
- 实验性工具

页面内容不涉及任何特定商业或营销目的，仅作为技术存档与共享使用。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 也可通过 GitHub Pages 部署后在线访问。
